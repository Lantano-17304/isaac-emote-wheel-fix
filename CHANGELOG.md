# Changelog

## v0.1.0-pre.3

首个公开预发布版本，仅支持 Steam 原版《The Binding of Isaac: Repentance+》
`1.9.7.17.J460`（Windows 10/11、Xbox/XInput 控制器）。

### 已知限制

- 当前只有 Xbox/XInput 控制器确认可用。
- 已有反馈表明部分 PlayStation、Nintendo Switch、DirectInput 和其他通用手柄
  无法被游戏映射到补丁使用的右摇杆组，本版本不保证兼容。
- 不建议把右摇杆映射成左摇杆；这会重新耦合人物移动和表情选择。

### 功能

- R3 打开原生联机表情轮。
- 右摇杆选择表情。
- A 或 R3 确认发送。
- 左摇杆始终只控制人物移动，不影响表情选择。
- 表情轮开启期间保留右摇杆的原有射击输入。
- 提供零额外运行库依赖的小型图形化安装与回滚管理器。

### 安全边界

- 不修改 `isaac-ng.exe`。
- 不拦截人物移动、射击或网络输入包。
- 未识别的游戏版本默认仅诊断，不安装活动补丁。
- 运行时签名或结构校验失败时保持原版输入。

### 安装

1. 退出游戏。
2. 将压缩包中的文件夹解压到游戏目录。
3. 运行管理器，点击“安装并启用修复”。
4. 启动游戏并测试。

> 这是未签名原生 DLL 的预发布版本，安全软件可能产生误报。请只从本仓库
> Releases 下载，并核对 SHA-256。
